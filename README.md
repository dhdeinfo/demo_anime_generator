# Script Web Scraping untuk Pembelajaran dan Pengembangan

Script ini dibuat sebagai sarana pembelajaran mengenai teknik web scraping, pengolahan data, dan otomatisasi pengambilan informasi dari halaman web yang tersedia secara publik.

Tujuan utama dari script ini adalah membantu developer, pelajar, dan peneliti memahami cara kerja pengambilan data dari sebuah website, mulai dari proses request, parsing HTML, ekstraksi data, hingga penyusunan hasil dalam format yang lebih terstruktur.

## Tujuan Penggunaan

* Mempelajari teknik web scraping dan data extraction.
* Memahami struktur HTML dan cara membaca elemen halaman web.
* Belajar mengolah data hasil scraping menjadi format JSON atau format lainnya.
* Mengembangkan kemampuan automasi untuk kebutuhan riset dan eksperimen.

## Disclaimer

Script ini disediakan hanya untuk tujuan edukasi dan pembelajaran. Pengguna bertanggung jawab penuh atas penggunaan script ini. Pastikan penggunaan script mematuhi:

* Syarat dan ketentuan website yang menjadi sumber data.
* Kebijakan robots.txt yang berlaku.
* Hak cipta dan hak kepemilikan data milik pihak terkait.
* Peraturan perundang-undangan yang berlaku di wilayah masing-masing.

Penulis tidak bertanggung jawab atas segala bentuk penyalahgunaan script ini, termasuk namun tidak terbatas pada pengambilan data secara berlebihan, pelanggaran hak cipta, atau aktivitas yang dapat merugikan pihak lain.

## Etika Penggunaan

Gunakan script ini secara bijak dengan:

* Membatasi frekuensi request agar tidak membebani server target.
* Menghormati hak pemilik website dan kontennya.
* Tidak menggunakan data yang diperoleh untuk aktivitas ilegal atau merugikan pihak lain.
* Memberikan atribusi yang sesuai apabila diperlukan.

Dengan menggunakan script ini, Anda dianggap telah memahami dan menyetujui seluruh ketentuan di atas.
